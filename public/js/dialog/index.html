Jquery 弹出框

简单使用说明

页面头部包含下列JS：

<script language="javascript" src="jquery.min.js"></script>
<script language="javascript" src="drag.js"></script>
<script language="javascript" src="dialog.js"></script>

路径和图片设置：（在dialog.js头部设置）

var DIALOGIMGPATH = '/dialog/' //dilog所在路径;
var DIALOGBGIMG = '/dialog/tbtbg.gif'; //背景图片
var WINDOWIMG = '/dialog/window.gif'; //小图标
var BUTTONIMG = '/dialog/toolbutton.gif'; //关闭按钮
var CONFIRMIMG = '/dialog/icon_query.gif'; //询问框小图标
var ALERTIMG = '/dialog/icon_alert.gif'; //确认框小图标
var CANCELBUTTONIMG = '/dialog/cancelbutton.gif'//取消图标
var OKBUTTONIMG = '/dialog/okbutton.gif'//取消图标

提示框

Dialog.alert(msg,func1,time,w,h)  点击此处弹出提示框
msg 提示框出现的信息内容
func1 = null 点击确定按钮后的回调方法，未设置则关闭提示框
time = 0 到少毫秒后自动调用回调方法，如果未设置回调方法，则关闭提示框
w = 300 提示框宽度(像素)
h = 120 提示框高度(像素)

询问框

Dialog.confirm(msg,func1,func2,w,h)  点击此处弹出询问框
msg 弹出框出现的信息内容
func1 = null 点击确定按钮后的回调方法，未设置则关闭弹出框
func2 = null 点击取消按钮后的回调方法，未设置则关闭弹出框
w = 300 弹出框宽度(像素)
h = 120 弹出框高度(像素)

弹出框内嵌网页

Dialog.frame(id,url,title,w,h,m)  点击此处弹出
id 弹出框的ID号
url 网页地址
title 弹出框的标题
w 弹出框宽度(像素)
h 弹出框高度(像素)
m 模态

内嵌HTML内容

Dialog.html(id,html,title,w,h,m)  点击此处弹出
id 弹出框的ID号
html 弹出框的HTML内容
title 弹出框的标题
w 弹出框宽度(像素)
h 弹出框高度(像素)
m 模态

内嵌DOM对象内容

这个是DIV内容，只能使用一次
Dialog.element(id,elementID,title,w,h,m)  点击此处弹出
id 弹出框的ID号
elementID Dom对象的ID号
title 弹出框的标题
w 弹出框宽度(像素)
h 弹出框高度(像素)
m 模态

关闭弹出框

Dialog.close(id)  点击此处弹出
id 要关闭的弹出框的ID号
